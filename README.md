### Auto-Collect Actions
This feature handles gathering resources throughout the neighborhood. Activate this by pressing F2. Your character moves through predefined paths to pick up items. Ensure your path remains clear of obstacles to maintain smooth movement.

You manage the distance at which the script detects objects. Access the Range Settings menu by pressing F3. A slider allows you to increase or decrease the detection radius. Larger ranges cover more ground but may cause performance drops on older computers.

The overlay shows icons that represent your current status. A green square confirms the script connects to the game. A red triangle warns you if the script stops working. Keep an eye on these cues to monitor your collection success rates.

## 🛠️ Troubleshooting

If you encounter issues, follow these steps to resolve them:

* Script not detected: Restart both the script and the Roblox client. Ensure you run the script as an administrator.
* Game lag: Reduce the detection range in the settings menu. Lowering the resolution of the game window often helps.
* Keyboard inputs not working: Check if you have other software open that uses the same hotkeys. Close those programs to avoid conflict.
* Anti-virus alerts: Some security software flags automation scripts as suspicious. Add an exclusion for the script file in your anti-virus settings to prevent it from blocking the connection.
